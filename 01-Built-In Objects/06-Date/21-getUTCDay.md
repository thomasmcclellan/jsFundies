# `Date.prototype.getUTCDay()`

```js
const date1 = new Date('August 9, 2019 09:00:00 GMT+11:00')
const date2 = new Date('August 9, 2019 09:00:00 GMT-11:00')

console.log(date1.getUTCDay()) // 4 (Thursday (4), 08 Aug 2019 22:00:00 GMT)
console.log(date2.getUTCDay()) // 5 (Friday (5), 09 Aug 2019 20:00:00 GMT)
```

---

## Syntax:
`dateObj.getUTCDay()`

## Return value:
A `number` (specially an integer) corresponding to the day of the week for the given date, according to universal time:

| Value | Day |
|---|---|
| 0 | Sunday |
| 1 | Monday |
| 2 | Tuesday |
| 3 | Wednesday |
| 4 | Thursday |
| 5 | Friday |
| 6 | Saturday |

---

## Description:
The `getUTCDay()` method returns the day of the week in the specified date according to universal time, where 0 represents Sunday.
