# `Object.assign()`

```js
const target = { a: 1, b: 2 },
      source = { b: 4, c: 5 },
      returnedTarget = Object.assign(target, source);

console.log(target); // { a: 1, b: 4, c: 5 }
console.log(returnedTarget); // { a: 1, b: 4, c: 5 }
```

---

## Syntax:
`Object.assign(target, ...source)`

* **target**: the target `object`--what to apply the sources' properties to, which is returned after it is modified
* **source(s)**: the source `object`(s)--`objects` containing the properties you want to apply

## Return value:
The target `object`.

---

## Description:
The `Object.assign()` method copies all [enumerable own properties]() from one or more _source `object`_ to a _target `object`_.  It returns the target `object`.
