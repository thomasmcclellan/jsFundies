# `Math.cbrt()`

```js
console.log(Math.cbrt(-1)); // -1
console.log(Math.cbrt(1)); // 1
console.log(Math.cbrt(Infinity)); // Infinity
console.log(Math.cbrt(64)); // 4
```

---

## Syntax:
`Math.cbrt(x)`

* **x**: a number

## Return value:
The cube root of the given number.

---

## Description:
The `Math.cbrt()` method returns the cube root of a number.

  > Becaue `cbrt()` is a static method of `Math`, you always use it as `Math.cbrt()`, rather than as a method of a `Math` object you created (`Math` is not a constructor).
