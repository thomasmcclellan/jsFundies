# `Map.prototype.clear()`

```js
const map1 = new Map()
map1.set('bar', 'baz')
map1.set(1, 'foo')

console.log(map1.size) // 2

map1.clear()

console.log(map1.size) // 0
```

---

## Syntax:
`myMap.clear()`

---

## Description:
The `Map.prototype.clear()` method removes all elements from a `Map` object.
